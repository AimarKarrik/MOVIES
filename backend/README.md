# Backend for Movies App

This is the backend for the Movies App.

## Environment setup

### Prerequisites

- Node.js

### Install dependencies

```bash
npm install
```

### Run

```bash
npm start
```

## Endpoints

### Movies

| Method | Endpoint | Paramaters                     | Description      |
| ------ | -------- | ------------------------------ | ---------------- |
| GET    | /movies  | page: number   pageSize: number | Get movies paged |
